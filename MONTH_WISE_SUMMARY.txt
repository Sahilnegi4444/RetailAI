✅ MONTH-WISE SALES CHART - FEATURE COMPLETE

📊 What Was Added:
   ✓ Month-wise bar chart in Dashboard Overview
   ✓ Shows last 3 years of data
   ✓ Year selector dropdown
   ✓ All 12 months displayed
   ✓ Dark theme styling
   ✓ Responsive design

🎯 How It Works:
   1. Automatically detects all years in database
   2. Shows only last 3 years (e.g., 2024, 2025, 2026)
   3. Default selection = most recent year
   4. User can select any year from dropdown
   5. Chart updates with month-wise data
   6. Shows "No data" for months with zero sales

📈 Chart Features:
   ✓ Blue bar chart with professional styling
   ✓ All 12 months (Jan-Dec)
   ✓ Sales volume on Y-axis
   ✓ Month names on X-axis (rotated for readability)
   ✓ Hover tooltip showing exact values
   ✓ Responsive on all screen sizes

🔧 Technical Details:
   ✓ Uses Recharts BarChart component
   ✓ Data calculated with useMemo (efficient)
   ✓ Safe number conversion (no NaN)
   ✓ Aggregates sales from all items
   ✓ No additional API calls
   ✓ Smooth state management

📍 Location:

   "📊 Month-Wise Sales (Last 3 Years)"

✨ Example Scenarios:

   Scenario 1: Database has 2024, 2025, 2026 data
   → Shows: 2024, 2025, 2026 (last 3 years)
   → Default: 2026
   → User can select: 2024, 2025, or 2026

   Scenario 2: Database has 2022, 2023, 2024, 2025, 2026 data
   → Shows: 2024, 2025, 2026 (last 3 years only)

   → Chart displays bars only for months with data
